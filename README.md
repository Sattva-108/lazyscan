## Installation

1. Download the ZIP.
2. Extract and copy the `lazyscan` folder to:

> World of Warcraft/Interface/AddOns/

> If you downloaded from "Latest source", the folder may be named `lazyscan-master`. Rename it to `lazyscan`.
3. Restart the game.
4. Open settings with **minimap icon** or `/lazyscan` or `/lscan`.

## Commands

| Command | Action |
|---------|--------|
| `/lazyscan` or `/lscan` | Toggle settings panel |
| `/lazyscan toggle` | Toggle scanning on/off |
| `/lazyscan start` | Start scanning |
| `/lazyscan stop` | Stop scanning |

## Features

- **Scans minimap** for ore and herb nodes **in real time**
- Disable nodes **you don't want to gather**
- Supports: **ElvUI Farm Mode, FarmHUD**
- Russian client support
